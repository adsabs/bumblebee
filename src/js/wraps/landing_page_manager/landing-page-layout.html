What is Bumblebee?

This new beta interface, codename "Bumblebee", offers a new way to search ADS: rather than searching only titles and abstracts, like "ADS Classic", Bumblebee allows you to search the full text of articles as well. For more information about how to create powerful searches click here.

How can I collaborate?

All of Bumblebee's code is publically accessible at Github.

You can programmatically access ADS Data by making an account with ADS and signing up for an API key. For more information about the API, check out the API README.

Where can I find...?

You can still access ADS Classic and ADS 2.0.

To learn more about this new interface and all of its functionality, visit the new ADS help pages.
