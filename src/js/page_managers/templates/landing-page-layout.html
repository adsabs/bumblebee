A new way to search the

Astrophysics Data System

What is Bumblebee?

Bumblebee is a new interface for the ADS, featuring a clean new look and easier search and filtering operations. New features are being added all the time.

How can I collaborate?

All of Bumblebee's code is publically accessible at Github. Information about signing up for an ADS developer API key will be available shortly.

Where can I find...?

You can still access both ADS Classic as well as ADS 2.0.

For questions or comments, contact adshelp (at) cfa.harvard.edu
